# Reshade-keybind-toggle-remover-BETA-
A lightweight, zero-dependency Windows batch script (.bat) designed to strip keyboard shortcut hotkeys (key toggles) from ReShade .ini preset files.
# How to Use
Download the remove_toggles.bat file from this repository.

Find the ReShade preset file (.ini) you want to clean.

Drag and drop your .ini file directly onto the remove_toggles.bat file.
